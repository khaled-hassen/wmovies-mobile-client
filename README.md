A mobile app client for [wmovies](https://github.com/khaled-hassen/wmovies) made with:
* Typescript
* Apollo GraphQL
* React native + Expo

## Getting Started

1. clone the repo

```bash
git clone https://github.com/khaled-hassen/wmovies-mobile-client.git
cd wmovies-mobile-client
```
2. setup the app
```bash
# dir: wmovies-mobile-client/
npm install
# or
yarn install
```
3. start the app
```bash
#dir: wmovies-mobile-client/
npm start
# or
yarn start
```
### Building the app
* android
```bash
# dir: wmovies-mobile-client/
npm run android
# or
yarn android
```
* ios
```bash
# dir: wmovies-mobile-client/
npm run ios
# or
yarn ios
```
